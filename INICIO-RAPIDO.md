## 🐛 PROBLEMAS?

### Site não abre?
Verifique se a porta 3000 está livre:
```powershell
npm run dev -- -p 3001
```

### Erro ao instalar?
Limpe e reinstale:
```powershell
Remove-Item -Recurse -Force node_modules
npm install
```

### Mudanças não aparecem?
Pare o servidor (Ctrl+C) e inicie novamente:
```powershell
npm run dev
```
